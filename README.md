# 专 专爪转 驻专拽

## 专砖转 拽转

- **Node.js** (专住 18 注)
- **npm**

---

##  专爪转 拽

```bash
cd backend
npm install
npm run dev
```

砖专转 专抓 注 `http://localhost:3000`

**注专:**  砖拽抓 `backend/.env`  转 砖转:
- `PORT=3000`
- `SUPABASE_URL=your_supabase_url`
- `SUPABASE_KEY=your_supabase_key`

---

##  专爪转 驻专

```bash
cd frontend
npm install
npm start
```

驻拽爪 转专抓 注 `http://localhost:4200`
